Introduction

Jekyll is a simple static site generator that can be set as a landing page or an intro for those getting used to web development. You can work with Jekyll environments using a markup language such as HTML and Markdown. Your website can be hosted on Github, avoiding the costs for storing your website on a server. As you learn Jekyll, you will be able to customize your site with original layouts or those located in the Jekyll themes.

Ruby Command Prompt

Although the first installation of Ruby is complete, there are a few extra steps to follow once initial installation is complete. A command prompt screen will appear when running Ruby with the various options:

  • For best installation purposes, its best to go through options 1, 2, and then 3.
  • Once option 3 is complete, close out of the application.

Command Line Check

Next you want to check for any updates as well as if Ruby is installed correctly. For this section, you can either use the Windows command line propt or Git bash. The example provided is using git bash.

  • To check the version of Ruby installed, type the following command: ruby -v
  • For the Gem package installation, type the following command: gem -v.
